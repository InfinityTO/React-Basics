# 3 Methods Used to define a Functional Component

You can use either of the below defined methods to create functional components in React

1. Function Declaration
2. Function Expression
3. Arrow Function


## 1. Function Declaration

```
function Nav (props) {
	return (
		   <ul>
			<li> {props.first} </li>
		   </ul>
	)
}
```

## 2. Function Expression
